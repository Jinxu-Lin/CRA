# REFINE 轮第 2 轮综合评估

**辩论类型**: 证据驱动精炼轮 -- 跨视角综合评估
**评审基准**: Pilot 结果 (N=100, Pythia-1B, DATE-LM) + 第 1 轮交叉批评
**评估日期**: 2026-03-17

---

## 一、跨视角共识点

经过两轮辩论，6 个视角在以下核心判断上达成一致：

### 共识 1：FM2 评测协议失效是全规模实验的最高优先级问题

所有视角均承认：rank 指标（AUPRC、R@K、MRR）对均值平移数学上不变，pilot 中 12 个 method-task 组合零增益不能被解读为"FM2 不存在"。Empiricist 最先明确诊断为"评测协议设计缺陷"，Contrarian 从反面佐证（FM2 若不改变排序，则不是排序任务的缺陷），Pragmatist 早在 pilot 前就预警了任务条件均值减法的必要性。全规模实验**必须引入连续指标**（Kendall tau、Spearman rho、LDS 连续版本）。

### 共识 2：毒性任务逆转定义了 CRA 的适用边界

TRAK (0.926) > RepSim (0.685) 的 24pp 逆转被所有视角接受为系统性现象而非噪声。当归因信号主要来自梯度范数（毒性检测中 Raw Dot IF 无 Hessian 已达 0.94 AUPRC，Cohen's d=2.66），FM1 叙事不适用。这是合理的范围限定，论文中必须明确表述。

### 共识 3：H7 在 pilot 规模的失败是工程限制，非理论否证

N/d=0.049 导致协方差矩阵欠定，所有视角认同此诊断。分歧仅在于全规模（N/d~2.7）时能否恢复。PCA 降维白化（N/k >> 1 时 k=64/128）是公认的最有希望出路。

### 共识 4：TRAK-PCA@k=d 与 RepSim 的 30.8pp 缺口是核心挑战

FM1（维度缺陷）是必要但不充分的解释。需要系统分解缺口来源。Empiricist 的残差分析（Exp 3.3）被多数视角认可为最佳分解方案。

### 共识 5：BM25 竞争力需全规模确认

counterfact pilot 规模下 BM25=1.0 可能是小样本词汇重叠效应，但不能假设——必须在 N=5473 全量下实验确认。Contrarian 进一步要求纳入专用检索模型（Contriever/GTR）作为更强 baseline。

### 共识 6：phi^T M psi 框架至少具有分类学价值

即便 Contrarian 质疑其预测能力，也承认该框架是首次对 5 个表示空间方法的系统统一，加上 DATE-LM 上的首次 2x2 factorial，本身即有贡献。争议仅在于"理论"还是"符号"的定位。

## 四、假设修订建议

### H2（对比评分不对称）

**现状**: pilot 中所有 12 个 method-task 组合增益恰好 0.0pp，因 rank 指标对均值平移不变。

**修订方案**:
1. **替换主指标**: 将 Kendall tau / Spearman rho 作为 H2 的主要评测指标，rank 指标降为辅助
2. **修订预测**: "对比评分在参数空间方法上的 Kendall tau 增益 >= 0.05，在表示空间方法上 < 0.02"（因表示空间 FM2 天然较低）
3. **补充机制验证**: 受控污染注入实验（Exp 2.2）作为 H2 的因果验证——若 alpha>0 时对比评分能恢复 alpha=0 性能，则 FM2 矫正机制有效
4. **考虑 OS-CFAR 变体**: 若全局均值减法在连续指标下仍效果有限，尝试中位数/截尾均值替代（Interdisciplinary R1），利用其打破 rank 不变性的特殊性质

**风险**: 若连续指标下对比评分增益仍 < 0.02，FM2 叙事需从"两个独立信号处理缺陷"降格为"FM1 主导 + FM2 为理论假设"

### H7（白化归因）

**现状**: 所有任务下降 8-11pp，N/d=0.049 协方差欠定。

**修订方案**:
1. **放弃全维度白化**: 在 DATE-LM 规模下（即使全量 N/d~2.7），d=2048 的全维度协方差逆仍然边缘可用。不再将全维度白化作为主要贡献声明
2. **改为 PCA 降维白化**: H7-revised 预测"在 PCA 子空间（k=64-128, N/k > 40）内白化优于 M=I"
3. **设定明确阈值**: 若 PCA 白化在任意 k 值、任意任务上均无法超过标准 RepSim >= 3pp，则在论文中诚实报告为阴性结果
4. **备选理论**: 若白化失败，考虑将 Theoretical 的 Proposition 2.2 重新解读——"各向异性时白化增益更大"的前提是协方差估计准确；在 LLM 规模下，数据量与维度的比值可能系统性地不足以支撑白化

**风险**: 即使 PCA 白化成功，增益可能仅 3-5pp，不足以支撑"N-P 最优检测器"的强声明

### H9（各向同性）

**现状**: 方向完全反转，rep_cond=3.1e10 >> grad_cond=3,589。但 N=100 < d=512，条件数不可靠。

**修订方案**:
1. **彻底放弃原始形式**: 不再声称"表示空间近各向同性（kappa~1）"
2. **替换为谱集中度指标**: 新 H9 改为比较表示空间 vs 梯度空间的有效维度比 r_eff/d，而非条件数。pilot 数据显示 rep r_eff(95%)=63/512=0.12 vs grad(full) r_eff=10/70M=1.4e-7，表示空间的信号分布确实更均匀（相对其名义维度而言）
3. **全规模验证**: 在 N=5000, d=512 (Pythia-70M) 下重测条件数，此时 N/d~10 足以可靠估计。若条件数降至 <1000，说明 pilot 的极端值确实是 N < d 导致的工件；若仍 >10^4，则各向异性是真实的，需要新的理论解释
4. **提出替代解释**: 若表示空间确实极度各向异性但 M=I 仍然有效，可能的解释是：(a) 归因信号与各向异性方向对齐，使得 M=I 恰好是"自然权重"；(b) 表示空间的各向异性是"有益的"——高方差方向恰好是高信息量方向

**风险**: 若全规模下各向异性被确认，Theoretical 的 Proposition 1.1 核心前提崩塌，需要全新的理论解释 M=I 有效性

## 五、必须放弃的方向

### 5.1 全维度白化 RepSim（H7 原始版本）

**放弃理由**: 在 DATE-LM 最大任务（N=10,187）和 Pythia-1B（d=2048）的条件下，N/d~5，仍处于 Ledoit-Wolf 估计的边缘地带。pilot 全面失败（-8 至 -11pp），且 Contrarian 正确指出即使全规模也是"结构性问题而非规模问题"。继续投入计算资源在全维度白化上是低回报高风险的。

**替代**: PCA 降维白化（k=64-128），在 N/k >> 40 的安全区间内操作。

### 5.2 过于乐观的框架预测声明

在全规模实验结果出来之前，论文中不应包含以下声明：
- "白化 CRA 是 phi^T M psi 族中 N-P 最优的"（依赖 H7 验证）
- "表示空间近各向同性使得 M=I 足够"（H9 已被否证）
- "FM1 和 FM2 是独立的信号处理缺陷"（H3 平凡满足，无统计功效）

**替代**: 将框架定位为"系统化分类学 + 可证伪的诊断假说"，以实验结果为权威。

### 5.3 IV 工具变量归因（Interdisciplinary E2）

**放弃理由**: 数据增强（同义词替换/改写）作为工具变量的排他性限制（"不影响任务特定信号但改变共同影响"）在 LLM 表示中极难验证。改写几乎必然改变语义内容（phi_task），违反排他性假设。第 1 轮批评已指出此问题，Interdisciplinary 未能给出充分的实验设计层面回应。

**替代**: 受控污染注入（Empiricist Exp 2.2）提供了更干净的因果验证，无排他性假设风险。

### 5.4 全规模 CATCL（在核心假设验证前）

**放弃理由**: CATCL 的平行趋势假设在表示空间极度各向异性时受威胁——微调前后的表示漂移将高度非均匀。在 H9 全规模走向明确、FM2 通过连续指标被确认为真实现象之前，不应将 CATCL 纳入全规模实验。

**条件性恢复**: 若优先级 1（FM2 验证）和优先级 4（缺口分解）的结果支持"非线性语义结构是主因"，CATCL 的时序差分方式获得额外动机，可在第二批实验中纳入。

### 5.5 sqrt(d) 谱维度预测（Interdisciplinary N3）

**降级为"讨论"材料，非实验目标**。pilot 的 RepSim PCA 饱和于 k=64（N=100 的自然截止）无法区分 sqrt(d)~45 效应和 N=100 限制。全规模实验中此测试可作为 Exp 1.2 维度扫描的副产品收集数据，但不作为独立实验目标。

## 六、对论文定位的综合建议

### 最可能的论文叙事（基于 pilot 证据和辩论共识）

**核心贡献应是实证性的，而非理论性的：**

1. **首次 DATE-LM 系统化 2x2 factorial**：{参数空间, 表示空间} x {标准, 对比评分}，在 3 个任务上展开，附带多方法锦标赛。这是文献空白的直接填补。

2. **FM1 的谱证据**：梯度协方差的极端低秩性（r_eff << d << B）、TRAK 维度扫描的饱和模式、以及 30.8pp 缺口的系统分解。这是 CRA 最强的实证贡献。

3. **phi^T M psi 作为组织框架**：统一 5+ 个表示空间方法的分类学，而非带强预测的理论。按方法的 phi/psi/M 配置组织实验结果，让数据说话。

4. **任务类型边界的发现**：毒性（梯度范数驱动）vs 事实归因/溯源（语义相似性驱动）的系统差异，是对 DATE-LM "no single method dominates" 发现的机制性解释。

5. **诚实报告的阴性结果**：H7 失败、H9 逆转、BM25 竞争力、30.8pp 缺口——这些增强论文可信度并定义范围。

### 最大风险场景

若全规模实验下：
- (1) Contriever/GTR 匹配 RepSim
- (2) PCA 白化在所有 k 值下均不优于 RepSim
- (3) 连续指标下对比评分增益 < 0.02

则 CRA 需从"信号处理诊断理论"降格为"系统化 benchmark + 分类学"。如 Contrarian 所言，这仍是有价值的贡献，但论文定位需根本性调整。

### 最值得期待的发现

受控污染注入（Empiricist Exp 2.2）和 TRAK-PCA 缺口分解（Empiricist Exp 3.3）是全辩论中评估最高的两个新实验设计。前者直接决定 FM2 的真实性，后者量化 FM1 的解释力上限。二者结果的组合将决定论文的最终叙事框架。

---

## 七、辩论协调者结语

经过两轮精炼辩论，CRA 项目的实验路径已从 pilot 后的不确定状态收敛为清晰的优先序列。核心判断如下：

**信心上调的部分**: FM1 谱证据（pilot 强支持）、2x2 factorial 的文献填补价值（确定性贡献）、毒性逆转作为范围限定的学术价值（新发现）。

**信心下调的部分**: phi^T M psi 的理论预测能力（3/3 独特预测失败或平凡满足）、白化归因的实用价值（需 PCA 降维才有希望）、FM2 的实证基础（尚未被任何实验检测到）。

**关键行动项**: 立即执行优先级 1（FM2 验证套件，~3 GPU 小时），其结果将决定后续 4 个优先级实验中哪些需要调整方案。这是"实验驱动理论"而非"理论驱动实验"的研究策略——在 pilot 证据已经提示多个假设存疑的情况下，这是最稳健的前进方向。
